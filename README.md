# MyChatBot
MyChatBot is a Streamlit-based AI chatbot that allows users to upload a PDF and ask questions about its content using LangChain and OpenAI.

## ✨ Features
- Upload and read PDF files
- Text chunking and embeddings
- Semantic search using FAISS
- AI-powered question answering
- Take the user question as input
- Retrieve the most relevant documents from the vector database using similarity search
- Pass the both language model and the retrieved documents to a question-answering chain to generate an answer to the user's question
- Generate the response and display it to the user

## 🛠 Tech Stack
- Python
- Streamlit
- LangChain
- OpenAI
- FAISS
